# ap-infrastructure-automation-resources
Public Resources for Infrastructure Automation

Hosts ARM templates to deploy the app services for the Ayuda Platform
We have five templates:
* WebApps.json -> App service without staging slot
* WebAppsWithStagingSlot.json -> App service with staging slot
* WebAppsWithStagingSlotAndAppInsights.json -> App service with staging slot and application insights
* AppInsights.json -> Application insight
* AppServicePlans.json -> App service plans
